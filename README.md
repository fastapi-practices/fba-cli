## 简介

`fba-cli` 是 [fastapi-best-architecture](https://github.com/fastapi-practices/fastapi-best-architecture) 的 CLI 工具，用于一键创建、配置、管理和运行 FBA 项目。提供交互式引导流程和插件市场，大幅简化项目初始化和日常开发操作

## 安装

```bash
npm install -g @yuworm/fba-cli
```

```bash
fba-cli --help
```
